📱 iOS Швидка команда

Просто завантаж готову команду на iPhone — вона вже налаштована на цю адресу сервера. Виставить гучність 100%, зробить запит до AURA, озвучить твої невиконані задачі голосом Siri українською, а потім відкриє додаток.

⬇️ Завантажити готовий Shortcut
На iPhone: тап відкриє додаток Швидкі команди. Якщо iOS попереджає про «непідписану команду» — натисни Додати без довіри (для цього треба один раз запустити будь-який власний шорткат, тоді в Налаштування → Швидкі команди з'явиться перемикач).
1Тест сервера
Перш ніж імпортувати команду, перевір що адреса вже відповідає:
2Автоматизація о 09:00 (щоденний будильник)
  1. Відкрий Швидкі команди → Автоматизації → +.
  2. Обери Час доби, встав 09:00, повторення щодня.
  3. Дія: Виконати команду → вибери щойно імпортовану AURA Discipline.
  4. Вимкни перемикач Запитувати перед запуском — тоді ранкова сирена спрацює сама.
3Або збери вручну (якщо iOS блокує імпорт)
Показати покроково
  1. Швидкі команди → +, назва Дисципліна.
  2. Дія Set Volume → 100%.
  3. Дія Get Contents of URL, URL — адресу з кроку 1, Method = GET.
  4. Дія Speak Text → джерело результат Get Contents, мова Українська.
  5. (Опційно) Дія Open URL → корінь PWA.
4Додаткові швидкі дії (опціонально)
Кожна — окрема команда з Get Contents of URL, Method = POST, Request Body = JSON.
  • 💧 Записати воду («Siri, я випив 250»)
    URL: /api/shortcuts/water
    Body: {"ml":250}
    У Shortcuts додай крок Ask for Input (Number) → підстав у Body як змінну {"ml":Provided Input}. Далі Speak Text → покаже message з відповіді сервера.
  • Швидка замітка → календар
    URL: /api/shortcuts/note
    Body: {"text":"Стоматолог","time":"14:30"}
  • Оновити вагу
    URL: /api/shortcuts/weight
    Body: {"weight":76.2}
  • Додати страву
    URL: /api/shortcuts/meal
    Body: {"name":"Протеїн","protein":30,"calories":120}
  • Позначити звичку
    URL: /api/shortcuts/toggle
    Body: {"habitId":"duolingo","done":true}
💡 Порада: Додай команду Дисципліна у Пункт керування (Settings → Control Center → додай «Швидкі команди»), щоб запускати аудит одним свайпом униз.
