📱 iOS ІНТЕГРАЦІЯ «ДИСЦИПЛІНА»

Швидка команда Дисципліна інтегрує ваш iPhone 12 з веб-додатком. Вона контролює ранковий підйом, озвучує невиконані вправи та дозволяє записувати швидкі замітки прямо в календар.

1. Завантажте Швидку Команду:
Назвіть завантажену команду строго Дисципліна на вашому iPhone.
Завантажити команду «Дисципліна»
2. Скопіюйте свій API Webhook URL:
Вставте це посилання всередину швидкої команди:
3. Налаштування Будильника (09:00):
Створіть автоматизацію в додатку "Швидкі команди":
  • Час доби: 09:00 щоденно.
  • Запуск: Негайно (без запиту).
  • Дія: Запустити команду «Дисципліна».
Siri озвучить ваші борги (зуби, м'юїнг, вакуум або навчання) та увімкне сирену при порушенні.
4. Додавання у Пункт Керування (Зверху):
Щоб запускати команду з верхньої панелі iPhone (де ліхтарик, інтернет):
  • Відкрийте Параметри (Settings) → Пункт керування (Control Center).
  • Додайте елемент «Швидкі команди» (Shortcuts).
  • Виберіть команду «Дисципліна» для відображення як віджета.
5. Додаткові швидкі команди iPhone:
Ви можете налаштувати кнопки швидкої дії на робочому столі або віджетах:
  • Швидка замітка: POST на /api/shortcuts/note
    Тіло: {"date":"РРРР-ММ-ДД", "text":"Ваш текст"}
  • Оновлення ваги: POST на /api/shortcuts/weight
    Тіло: {"weight": 76.2}
  • Швидкий протеїн/їжа: POST на /api/shortcuts/meal
    Тіло: {"name":"Протеїн", "protein":30, "calories":120}
  • Виконати звичку (Duolingo/М'юїнг): POST на /api/shortcuts/toggle
    Тіло: {"habitId":"duolingo", "done":true}
